# 🌸 Mood-Based Anime Recommendation Chatbot

A mood-based anime recommendation chatbot built with **Python**, **OpenAI LLM**, **Gradio**, and **Jikan API**.

This chatbot allows users to select their current mood, optionally enter extra preferences, and receive personalized anime recommendations. It also displays anime posters and trailer links.

---

## 📌 Project Overview

Many users spend too much time deciding what anime to watch. This chatbot helps solve that problem by recommending anime based on the user’s current mood.

The user:
- selects a mood using icon buttons
- optionally adds extra preferences
- clicks **Submit**
- receives 5 anime recommendations
- sees poster images and trailer links for the recommended anime

## 🧠 Chatbot Workflow

The chatbot works as follows:

1. The user selects a mood using an icon button.
2. The selected mood is stored in the application state.
3. The user optionally enters additional preferences.
4. The user clicks the **Submit** button.
5. Python sends a prompt to the LLM.
6. The LLM returns 5 anime recommendations.
7. The system extracts anime titles from the response.
8. Jikan API is used to fetch anime details such as:
   - poster image
   - synopsis
   - score
   - trailer link
9. The results are displayed in the Gradio UI.
